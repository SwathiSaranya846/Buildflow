# 📘 Full Stack Project 1 – Frontend Development

## 📌 Overview
This project is part of the Full Stack Development training program. It focuses on building the **frontend (user interface)** of a web application using HTML, CSS, and JavaScript.

The project demonstrates how to design responsive web pages and create interactive user interfaces.

---

## 🚀 Features
- Responsive web design  
- Clean and user-friendly interface  
- Interactive elements using JavaScript  
- Form handling (Login / Signup / Input forms)  
- Basic navigation between pages  

---

## 🛠️ Tech Stack
- **HTML5** – Structure of the web pages  
- **CSS3** – Styling and layout  
- **JavaScript** – Interactivity
